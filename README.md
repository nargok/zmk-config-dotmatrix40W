# dotmatrix40W - ZMK Configuration

このリポジトリは、40%サイズのワイヤレスカスタムキーボード「**dotmatrix40W**」用のZMKファームウェアリポジトリです。
このリポジトリをご自身のGitHubにコピー（Fork）することで、ブラウザ上で簡単にキーマップをカスタマイズできます。

## ⌨️ dotmatrix40W について

dotmatrix40Wは、使いやすさと機能性を詰め込んだコンパクトなワイヤレスキーボードです。

- **46キーのレイアウト**: 4行×12列をベースに、最下段は親指で押しやすい1.5Uサイズのキーを備えています。
- **ロータリーエンコーダ搭載**: 中央に配置されたエンコーダを回すことで、スクロールや音量・LEDの明るさ調整が直感的に行えます。
- **ワイヤレス対応 (Bluetooth)**: ケーブルなしでPCやスマートフォンと接続できます。
- **単三電池1本で動作**: 乾電池駆動に対応しており、自動スリープ機能によりバッテリーを長持ちさせます。
- **フィラメントLED**: 暖かみのあるバックライトを搭載し、明るさの調整も可能です。

## 🛠️ キーマップのカスタマイズ方法

[Keymap Editor](https://nickcoutsos.github.io/keymap-editor/) を使えば、専用のソフトをインストールすることなく、ブラウザ上で簡単にキー配置を変更できます。

### 1. リポジトリをコピー（Fork）する
画面右上の **Fork** ボタンを押して、このリポジトリをご自身のGitHubアカウントにコピーしてください。

### 2. Keymap Editorにアクセス
ブラウザで [Keymap Editor](https://nickcoutsos.github.io/keymap-editor/) を開きます。

### 3. GitHubと連携する
Keymap Editor上でGitHubアカウントを連携（Authorize）し、リストから先ほどForkした `zmk-config-dotmatrix40W` を選択します。

### 4. キーマップを編集する
画面上でドラッグ＆ドロップして、お好みのキー配置に変更します。
> **💡 エンコーダの設定について**
> ロータリーエンコーダ（回転操作）の割り当ては、画面内の **「Encoders」** タブから設定してください。レイアウト上の中央下段にある空白スペースは、システム仕様上のダミー枠です。

### 5. 保存してファームウェアをビルドする
編集が終わったら、Keymap Editor右上にある **Save** ボタンを押します。
自動的にGitHub上でビルド作業（ファームウェアの作成）が始まります。

### 6. ファームウェアを書き込む
ご自身のGitHubリポジトリの **Actions** タブを開き、完了したビルドからファームウェア（`.uf2` 形式のファイル）をダウンロードします。あとは、USB接続したキーボードにそのファイルをドラッグ＆ドロップすれば書き込み完了です！

## ⚙️ 初期のレイヤー設定

デフォルトでは以下の3つのレイヤーが設定されています（自由に書き換え可能です）。

- **Layer 0 (Base)**: 通常のタイピング用。エンコーダは「上下スクロール」。
- **Layer 1 (Fn & Number)**: F1〜F12キー、テンキー、記号入力。エンコーダは「左右スクロール」。
- **Layer 2 (System)**: Bluetooth設定のリセット（`BT_CLR`）や、LEDバックライトの輝度調整。エンコーダで明るさを変更できます。
